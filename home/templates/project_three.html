This is an responsive website created with Django, Python, JavaScript, HTML and CSS.

This website is a fullstack web application for an ecommerce business.


In assessing the needs of the user, it was determined that as a user, I would like to:

  • easily navigate to the information I need.
  • be able to specify the type of surfing I wish to view (group, lessson etc.).
  • find relevent information such as the tide.
  • have an easy format of contacting the business.
  • view the area and other customers in the gallery
  • learn about the business.
  • have any common queries answered and accesible.
  • find the business using google maps.
