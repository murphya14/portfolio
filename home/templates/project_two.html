This is an responsive website created with Django, Python, JavaScript, HTML and CSS.

This website is a fullstack web application for an ecommerce business that requires authentication and utilised as an advertising and sales platform.


In assessing the needs of the user, it was determined that as a user, I would like to:

  • have a site where I can buy a product specifically for a hobby.
  • be able to specify the hobby through category in order to filter the products.
  • understand how other consumers felt about the product.
  • buy off of a transparent company and so, be able to leave a review on my purchases.
  • be able to see more detail about the product (more than just the image and price.)
  • be able to view my cart and remove items if needed.
  • to be able to pay securely.
  • be able to register and log in to my personal account.
