Simon Says

View the project

This is an interactive website created with JavaScript, HTML and CSS.

The purpose of this website is to provide a challenging and engaging memory game that allows for an energetic user interface and captures the attention of the player. This game can be implemented in a learning capacity in many ways, pontentially as starter task in order to implement Gagne's 9 Events first step: gain attention.

To engage the user, I incorprated vivid colours and accompanied user actions with audio sounds. I also provided a detailed modal which contains an explaination of how to play. To maintain all users interest, there is the option of a more challenging game by using the "Strict Mode" setting. This allows the game to cater to all levels.


In assessing the needs of the user, it was determined that as a player, I would like to:

  • see a striking design that is visually stimulating and provides an incentive to play the game.
  • be able to access information so I can understand more about the game.
  • be able to control the power button so I can experience the interactive nature of the game.
  • be able to control the start button so as to begin when I am ready.
  • be able to set a challenge level.
  • be able to get instant feedback on whether I have lost/game over or gone ahead to the next level/win.

As part of designing the game, I created wireframes using Balsamiq
